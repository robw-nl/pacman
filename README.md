I started experimenting with a simple pacman maze, dots for pellets 
where one triangle (pacman) was followed by 3 circles (ghosts) coming 
from different angles. This simple experiment ended up becoming a 
fulltime, 2-months of hard work C project where I eventually started 
to use Gemini 3.1 Pro as a senior co-developer to fine-tune the code, 
making it error free and fixing some ghost AI faults that crept in. 
I'm still improving it but the game is fully stable, free of memory 
leaks, it has an auto demo mode (kicks in if the  user does not press 
enter to start). Has all the original sounds, sprites and more 
importantly, a true implementation of this original ghost AI. And it 
is hard, all the way from the get go. If you manage to reach level 256
you're even treated to the original game crash artifacts, after which 
you can continue. Be ready for a week of non stop playing for that!
 
For info on the gameplay and AI rules, these are good sources:
https://pacman30thanniversary.net/pacman-ghost-movement-pattern/ and 
https://aighost.co.uk/pac-man-ghost-ai-how-the-classic-games-enemies-think/
